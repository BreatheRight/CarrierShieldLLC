# Valora - Carrier Shield Compliance

A private pre-production demo website for a trucking compliance services company.

## Tech Stack
- React 18+ (Vite)
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)
- Framer Motion (Animations)

## Project Overview
This project is a lean service-site MVP designed for stakeholder demonstration. It transitions a previous single-page landing site into a multi-page routed application showcasing Valora's core service offerings: Remote Safety & Compliance, Permit management, and New Carrier Launch support.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
